# T题海搜题工作流

## 触发条件
用户在 `基于题海api的网测助手/` 放入题目后，执行以下流程。

## 答案优先级（严格执行）
**题库 > 本地题库 > 课本 > AI**
1. 题库命中 → 直接用，标注"题库"
2. 题库未命中 → 本地题库搜索 → 找到 → 标注"本地题库"+ 具体题号
3. 本地题库无 → 课本搜索 → 找到原文 → 标注"课本"+ 位置
4. 都没有 → AI 兜底 → 标注"AI"+ 推理依据
5. 全部完成后 → **AI 逐题核验**：查逻辑矛盾、选项错配、来源标注错误

## 工作流

### 1. OCR 提取（多方交叉验证）+ 纠错 + 选项合并
**至少 3 次 OCR**（不同对比度/锐度参数），取 2 次以上一致的高置信文本。
EasyOCR 增强对比度后逐字提取，然后**用课本词库自动纠错**：
- 从课本 md 提取所有 2-6 字高频词，构建纠错词库
- OCR 结果中每个词与词库比对，编辑距离 ≤1 的疑似错字自动替换
- 例：OCR"军制虫" → 词库有"军制史" → 自动修正

**选项跨行合并**：OCR 经常把选项字母和文本拆成两行（"B" 一行，"信息主导" 下一行）。解析时遇到单独出现的字母 A-E，自动与下一行合并为一个选项。

每道题拆出：题号、题目原文（纠错后）、类型、各选项文本。

### 2. 批量 API 搜题（优先，多轮降级搜索）
逐题调用 itihey API：
```
POST https://platform.itihey.com/v1/search
x-api-key: itk_56bdc146d6f6afb0a38959868fc75c728e20bd05b554ea8a
```
- 类型：0=单选 1=多选 2=填空 3=判断 4=简答
- **选项文本必须随题传入**，用 `|` 分隔，判断题固定传 `对|错`
- **多轮降级搜索**：
  1. 完整题 + 完整选项 → 搜
  2. 搜不到 → 去虚词（的/了/是/在/和/与/等），只留实词主干 → 再搜
  3. 再搜不到 → 切短词：提取最长 3 个关键词+最后 2 个选项关键词 → 再搜
  4. 还搜不到 → **换词重试**：用选项关键词反搜（用选项内容当搜索词反向匹配）
  5. 都没结果 → 进入课本兜底
- 返回 `use_ai: false` 且 `answer_string` 非空 → 标"题库"
- 返回 `use_ai: true` 或答案为空 → 进入课本兜底
- **多选/判断注意**：多选题取 `answer_string` 全部项；判断题传选项 `对|错`，从 `answer_string` 或 `answer_key`（A=对/B=错）推断
- **判断题兜底**：判断题型银行命中率天然偏低，如 API 返回 AI 但答案与课本/常识一致则直接采信
- **OCR 可信度标记**：OCR 文本明显碎片化时（单题文字被拆成 5 段以上），优先用选项关键词反向搜

### 3. 课本兜底
**仅 API 完全未命中时使用**。在目录中按文件大小（约571KB）定位课本 md，按关键词搜索。
- 找到明确原文 → 标"课本"
- 找不到 → 标"未知"，题干保留，答案和选项留空
- **注意**：课本是兜底不是裁判。API 有答案就不要拿课本去对比质疑。

### 4. Claude 最终核验
AI 逐题审核，**以题库为准**，重点检查：
- 来源标记是否正确（题库/AI/课本/未知）
- OCR 解析错误导致的遗漏（多选只取一项、选项跨行等）
- 判断题的"对/错"方向是否与选项匹配
- **不主动质疑题库答案**，仅在 API 完全未命中时才用课本辅助判断

核验结果追加到输出文件中：
- 有问题的题标注 `⚠️ 核验未通过` + 理由
- 无误的题标注 `✅`

### 5. 输出文件
生成 `D:\Documents\25262\测试N 答案.md`（N 对应测试文件夹名），**三部分结构**：

**主表格**：所有题目的快速速查

**📖 课本查证**：每道课本来源题列出原文 + 位置
> 课本原文："力争到**2035年基本实现国防和军队现代化**……"（专题一 第三讲）

**🤖 AI 兜底**：每道 AI 来源题给出推理逻辑
> 课本定义智能化战争为"人机主导、使能作战……"，选项"算法优势"最接近。

**规则**：
- "选项"列 = 字母 + 选项文字（如 `**D.22000**`），防试卷选项对调
- "答案"列 = 答案关键词
- "题目"列 = 完整题目
- "来源"列仅三类：题库 / 课本 / AI
- "核验"列：非题库的题标 ✅（已验证），题库的标 ✅
- 题库是主依据，课本/AI兜底题集中附详细说明

## 实战经验
- **标点决定命中**：《》（）""。等符号不能丢
- **换说法救题**："哪个不是"→"哪项不属于"；判断题补法律术语
- **补全残缺题文**：OCR 碎片导致不完整时，用常识补全再搜
- **多方 OCR 交叉**：至少 3 次不同参数，取高置信交集
- **无选项兜底**：带选项搜不到时，去选项用 type=0 重试
